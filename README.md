# Happy Birthday, Ryleigh 🏔️

A little birthday gift site: it oil-paints a **random famous Alberta sight** right in
the browser — the painting is rendered pixel-by-pixel in **Rust**, compiled to
**WebAssembly**. Refresh for a different wonder. The loading screen is a
roundhouse. 🥋

**Live:** https://sudo-kno3.github.io/happy-birthday-ryleigh/

## How it works

- `src/lib.rs` — a from-scratch Rust image engine:
  - `stylize(pixels, w, h, kick)` — an oil-painting filter (intensity-histogram
    "oil" smear + canvas-weave texture) applied to a photo; with `kick`, it draws
    the roundhouse cameo into the foreground.
  - `render(w, h, seed, kick)` — a bonus fully generative Rocky-Mountain scene
    (kept for later; not used by the current site).
- `dist/` — the static site (this is what gets hosted):
  - `index.html` — picks a random image, loads it to a canvas, calls the WASM
    `stylize`, shows the painting. Toggle photo/painting, save a PNG keepsake.
    The loading animation is an SMIL-animated SVG (her face as the head).
  - `img/` — five **public-domain / CC0** Alberta photos (Wikimedia Commons):
    Moraine Lake, Lake Louise, Peyto Lake, Drumheller Hoodoos, Vermilion Lakes.
  - `pkg/` — the compiled WASM + JS glue (generated by `wasm-pack`).

## Build

Requires the Rust toolchain, the `wasm32-unknown-unknown` target, and
[`wasm-pack`](https://rustwasm.github.io/wasm-pack/).

```bash
cargo test                                      # run the Rust unit tests
wasm-pack build --target web --out-dir dist/pkg --release
rm -f dist/pkg/.gitignore                        # so the wasm/js are deployable
```

Preview locally (WASM needs a server — it won't run from `file://`):

```bash
python3 -m http.server -d dist 8000    # then open http://localhost:8000
```

## Deploy for free (GitHub Pages)

The whole site is the `dist/` folder — host it anywhere static.

```bash
# publish dist/ to the gh-pages branch and enable Pages once:
git subtree split --prefix dist -b gh-pages
git push -f origin gh-pages
gh api -X POST repos/<owner>/<repo>/pages -f 'source[branch]=gh-pages' -f 'source[path]=/'
```

Or drag the `dist/` folder onto https://app.netlify.com/drop for an instant URL.
